Why SearchMaster?

Take your VS Code search capabilities to the next level with intelligent fuzzy search, configurable edit distance, and ranked results that help you find exactly what you're looking for.

Smart Fuzzy Search

Find files and content even with typos or partial matches using advanced fuzzy search algorithms.

Configurable Precision

Adjust edit distance to control how fuzzy your searches are, from exact matches to broadly similar terms.

Ranked Results

Results are automatically ranked by relevance, putting the closest matches first for faster navigation.

Installation Guide

Get SearchMaster up and running in VS Code in just a few steps

1

Download the Extension

Click the download button below to get the latest SearchMaster .vsix file.

Download search-master-0.0.0.vsix
2

Open VS Code Extensions Panel

Navigate to the Extensions view by clicking the Extensions icon in the Activity Bar.

Shortcut: Ctrl+Shift+X (Windows/Linux) or Cmd+Shift+X (macOS)
3

Install from VSIX

Click the three dots menu (‚ãØ) in the Extensions view and select "Install from VSIX..."

4

Select and Install

Browse to the downloaded .vsix file, select it, and click "Install". VS Code will install the extension automatically.

‚úì

Start Searching!

SearchMaster will appear in your sidebar. Click the SearchMaster icon to open the search panel and start using advanced fuzzy search.
